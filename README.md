BrickLib
========

This is a small library written in PHP to utilize LEGO brick lists.

Usage
-----

1. In LDD (LEGO Digital Designer) click on “Export BOM” and save as Excel file.
2. Open the Excel file and save as CSV file.
3. Run `bin/csv2xml [your saved csv file] > bricklink.xml`
4. Import the XML file on BrickLink
